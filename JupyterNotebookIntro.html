安装Jupyter

  1. 通过pip安装Jupyter
    pip install jupyter
  2. 安装Jupyter kernel
    python -m ipykernel install --user --name=tf00
    需要手动指定Jupyter的运行环境到当前虚拟环境下，否则它可能会使用系统默认的Python环境。

此处定义了一个二维数组，数组的第一维为None，在TensorFlow里定义为None表示一个可变长度的数组。
X = tf.placeholder(“float”, [None, num_input])
Y = tf.placeholder(“float”, [None, num_classes])
